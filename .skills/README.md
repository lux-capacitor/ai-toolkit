# Reusable AI Skills
This directory showcases how to create skills for common development workflows with AI agents. Invoke them via `/name` in Copilot or Claude Code.

**!~ IMPORTANT ~!** : Be sure you have run the Agent : `Agents / Skills Symlinker - Repo` in the root of this project so the skills symlink properly!

<br>

## Available Skills
| Skill | Description |
|--------|-------------|
| **sample-skill** | Example format to base skill creation on (can also use copilot's `/create-skill` command!) |

## Usage
Invoke prompts using the `/name` command in Copilot Chat:

```text
/<prompt-name> <extra context if desired or needed>
```

### Example
```text
/analyze-feature I'm trying to build a modal what components might I use?
```
