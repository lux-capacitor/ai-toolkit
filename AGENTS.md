# AI-TOOLKIT 
AI Toolkit for Shared Skills, Agents and workflow walkthroughs for getting the most out of our shared Github Copilot & Claude Code Usage. 

## INITIAL SETUP
When initializing with this repo, you need to check that the .claude/skills folder is symlinked to .skills and the .claude/agents folder is symlinked to .agents. If it is not you need to run the skill found at .skills/symlink-repo/SKILL.md
